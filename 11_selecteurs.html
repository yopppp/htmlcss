Aide mémoire des sélécteurs CSS déjà vus

  • * selection TOUS les éléments de la page html
  • .ma-jolie-nav selectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id séléctionne tous les éléments ayant le nom de mon id "mon-unique-id"

Propriétés CSS utilisées dans cette page


Décrivez leur usage et leurs valeurs possibles

  1. background: couleur du fond de l'élément
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. height: hauteur de lélément
  7. left: position à partir de la gauche de la fenêtre
  8. line-heinght: hauteur de l'interligne
  9. margin: marge extérieurs
  10. padding: marges intérieurs
  11. position: position de l'éléments dans la page (on n'a pas utilisé que la valeur (fixed) pour l'instant)
  12. text align: alignement du texte
  13. text-décoration: changement des valeurs héritées (ici le souligné qui est standars pour les liens sous tout les navigateurs est supprimés
  14. top: position à partir du haut de la fenêtre
  15. transition: temps de transition lors lors d'une animation (ici sur les liens de la barre de menu)
  16. width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative(%, em, ...)
